# Portfolio_Data_Science
Portfólio para Ciência de Dados e áreas correlatas.
Neste projeto, abordaremos um projeto dividido em 2 etapas: A primeira etapa será apresentado um notebook feito em python para estruturar dados, oriundos de um software de química quântica, em formato de DataFrame para ser utilizado na análise de dados. E a segunda etapa será feito um modelo de classificação utilizando modelos de machine learning com os dados do DataFrame da primeira etapa.
